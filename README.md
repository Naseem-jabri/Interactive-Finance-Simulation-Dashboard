# Financial Dashboard Project

I developed an interactive Financial Dashboard using Power BI to analyze historical performance and simulate future growth.



##  Key Insight Found

Despite achieving total sales of $118.7M, profitability was inconsistent. A significant portion of transactions was flagged as "Under Review" due to profit margins falling below 10%.


##  What the Data Revealed

- **High-Cost Clusters:** Certain products and segments are struggling with high COGS, leading to "leaking" profits.

- **Geographic Dominance:** Sales are concentrated in specific countries, but high volume doesn't always equal high profit margin.

- **Scenario Impact:** Small reductions in operational costs (COGS) have a massive exponential impact on the final Net Profit.



##  Actionable Solutions Based on Data

- **Cost Optimization:** Implemented a What-If Simulation to identify the exact cost reduction needed to hit profit targets.

- **Strategic Filtering:** Used interactive Slicers to isolate underperforming segments and target them with efficiency-boosting plans.

- **AI Forecasting:** Leveraged Power BI's forecasting to predict the next 6 months' trends, allowing for better inventory and budget planning.



##  Project Overview

A 2-page Power BI Dashboard designed for financial analysts. Page 1 focuses on Descriptive Analytics (Historical Data), and Page 2 focuses on Prescriptive Analytics (Predictive Modeling & Scenarios).



##  Tools & Technologies

- **Excel:** Data Cleaning and initial KPI formulation.

- **Power BI:** Data Modeling, DAX (Simulated Profit Measure), and Visualization.

- **DAX (Data Analysis Expressions):** Created a dynamic Simulated Profit measure linked to a Numeric Parameter Slider.



##  Features & Functionality

- **Dynamic Navigation:** Seamless sidebar navigation between Home and Forecast pages.

- **What-If Simulation:** A slider allowing users to adjust cost percentages and see real-time profit changes (e.g., reaching $1.15bn profit).

- **AI Forecast:** Automated trend analysis for future sales forecasting.



##  How to Use

1. Download the `.pbix` file.
2. Open in Power BI Desktop.
3. Interact with the Cost Reduction % slider on the Forecast page to explore different financial scenarios.
